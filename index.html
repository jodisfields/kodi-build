Jodisfields Kodi Repository

Repository for custom Kodi build with Fentastic, FenLight, and CocoScrapers.

repository.jodisfields-1.0.0.zip

Installation Instructions

  1. Open Kodi and go to Settings → File Manager
  2. Select Add Source and enter:
    https://jodisfields.github.io/kodi-build/
  3. Name it jodisfields and click OK
  4. Go back to Settings → Add-ons → Install from zip file
  5. Select jodisfields → repository.jodisfields-1.0.0.zip
  6. Wait for "Add-on installed" notification
  7. Go to Install from repository → Jodisfields Repository
  8. Select Program add-ons → Jodis Build Wizard → Install
  9. Run the wizard from Program add-ons to install the build
